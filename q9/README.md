### 🛠️ Alternative Method: Console Script
If you prefer to run the solver directly in your browser console, follow these steps:
##### 1. Reset Your Session:
If you are stuck or out of moves, use the "labyrinth_reset.js" script. Paste it into your console to start fresh.
##### 2. Run the Solver
Open the "labyrinth_solver.js" file.
Copy the entire code.
Go to the TDS Labyrinth Game Page.
Open Developer Tools (Right-click > Inspect > Console).
Paste the code, edit your `EMAIL` and `WEEK` at the top, and press Enter.
##### 3. Verification
Once the script finishes, it will display your JWT Token. Copy this and submit it to the TDS portal.
